Servidor Odin

Este servidor contiene mis desarrollos en pruebas, experimentos de sistemas como paneles o integración continua entre otros servidores.

También contiene apartados para depuración y monitorización de las aplicaciones web y apis de forma que puedo ir revisando resultados en el tiempo, depurando el código de estas.

Todo lo que encuentras aquí está bastante abierto al mundo y no tiene un valor real como podría tener otro servidor en producción.

Integración Contínua GoCD

Para la integración y despliegue automático de mis aplicaciones, utilizo esta herramienta para realizar el despliegue contínuo sin interacción humana.

Las validaciones de código he decidido realizarlas manualmente protegiendo la rama principal en cada repositorio. Sin embargo pruebas automatizadas también las realizo con esta herramienta.
